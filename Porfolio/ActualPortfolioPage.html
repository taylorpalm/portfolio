Javascript Polygon

Angular Matrix

This application generates a matrix of the given number of rows and columns with random integers from 1 to 100. It displays the matrix, transpose and the 2 added together if it is a square matrix.

React Sorted Table

This table pulls API data on earthquakes, formats it into the table and orders it by magnitude. The search function allows you to search and filter by location.
